## 📖 關鍵概念

| 概念 | 描述 |
|------|------|
| 原則主義 (Principlism) | 依賴抽象倫理原則進行決策 |
| 形式主義 (Formalism) | 機械應用倫理規則 |
| 倫理多元主義 | 不同文化對倫理問題有不同的切入點 |
| 知情同意 | 數據使用需獲得明確同意 |
| 算法透明 | 決策過程應可解釋和追溯 |

## 🌏 東西方視角

### 歐盟傳統
- 康德義務論 → 普遍適用原則
- 人權框架 → 個人權利優先
- 預防原則 → 先考慮風險

### 中國傳統
- 儒家集體主義 → 群體和諧優先
- 和諧社會 → 穩定壓倒一切
- 實用主義 → 發展與倫理並重
