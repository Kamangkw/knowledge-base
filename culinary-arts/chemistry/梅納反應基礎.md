## 核心概念

### 梅納反應（Maillard Reaction）
法國化學家Louis-Camille Maillard於1912年首次描述嘅化學反應。

**定義**：含羰基化合物（還原糖）與氨基酸在高溫下反應，產生數百種風味化合物和褐色色素。

**溫度**：通常在140-165°C（280-330°F）開始進行
**關鍵反應物**：
- 還原糖（葡萄糖、果糖）
- 氨基酸（尤其係游離氨基酸）
- 水份（適量水份加速反應）

### 化學機制（簡化版）

1. **初始階段**：糖類與氨基酸縮合形成N-取代糖胺
2. **高級階段**：通過Amadori重排產生更多中間體
3. **最終階段**：形成類黑精（melanoidins）——褐色色素
4. **風味形成**：同時產生嘅揮發性化合物（pyrazines, furans等）

### 影響因素

| 因素 | 效果 |
|------|------|
| 溫度 | 越高越快，但太高會焦化 |
| pH值 | 弱鹼性（~pH 8）加速反應 |
| 水份 | 適量水份有助反應物移動 |
| 糖類種類 | 還原糖比蔗糖更易反應 |
| 氨基酸種類 | 含硫氨基酸（半胱氨酸）產生更多肉香 |

## 為何重要

梅納反應係烹飪中最重要嘅風味來源之一：
- **烤麵包**嘅金黃色同香味
- **煎牛排**嘅焦香
- **烘咖啡**嘅複雜香氣
- **烤雞**嘅金黃表皮

冇梅納反應，食物會係蒼白且味道平淡。

## 實際應用

### 廚房技巧
1. **高温乾熱**：煎、烤、炸最能觸發梅納反應
2. **加鹼**：烹飪前加少許鹼（小蘇打）提升pH，加速反應且顏色更深
3. **避免過度**：控制時間，預防焦化（焦化係另一個化學過程，產生苦味）
4. **濕度控制**：肉類煎之前保持表面乾爽，水份會阻礙梅納反應

### 科學道理
- 牛排高溫煎完後休息——內部溫度會繼續上升（ residual heat）
- 醬油有助肉類表面上色——含有游離氨基酸同糖
- 烤蔬菜前加少許油——幫助熱量傳導但水份控制係關鍵

## 相關化學反應

### 焦糖化反應（Caramelization）
- 梅納反應常被混淆嘅另一個反應
- 焦糖化：糖類自己熱分解，不需要氨基酸
- 溫度更高（160-180°C）
- 產生焦糖特有甜味，而非肉香/烘烤香

### 美拉德反應產物（MRPs）
- 具有抗氧化性質
- 部分研究顯示可能有健康影響（仍在研究中）
