## 重要概念

| 概念 | 說明 |
|------|------|
| 梅納反應 | 加熱時糖與氨基酸的反應，產生焦香 |
| 焦糖化 | 糖類熱分解，產生甜味和顏色 |
| 乳化 | 油水混合的穩定狀態 |
| 水解 | 蛋白質/脂肪分解，軟化食材 |
