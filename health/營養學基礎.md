# 營養學基礎

## 基本資訊

| 項目 | 內容 |
|------|------|
| 名稱 | 營養學（Nutrition Science）|
| 研究對象 | 食物如何支持生命和健康 |
| 核心問題 | 什麼飲食模式最能促進健康長壽？ |
| 主要飲食模式 | 地中海飲食、MIND飲食、純素、生酮 |

## 核心內容

### 1. 巨量營養素（Macronutrients）

#### 蛋白質（Protein）
- 功能：建造和修復組織、酶、激素、免疫系統
- 每日建議攝入量：體重每公斤 0.8-1.2 克
- 优质来源：雞胸肉、魚類、豆類、雞蛋、乳製品
- 蛋白質不是越多越好——過量可能增加腎臟負擔

#### 碳水化合物（Carbohydrates）
- 功能：主要能量來源（大腦首選燃料）
- 分类：
  - 簡單碳水（糖、白麵包）→ 快速升高血糖
  - 複雜碳水（全穀物、蔬菜）→ 穩定能量
- 建議：以高纖維、低升糖指數（GI）的碳水為主

#### 脂肪（Fat）
- 功能：細胞膜組成、激素製造、維他命吸收
- 類型：
  - 飽和脂肪（動物脂肪）→ 適量
  - 不飽和脂肪（橄欖油、魚油）→ 推薦
  - 反式脂肪（加工食品）→ 避免
- Omega-3 脂肪酸：抗炎、益腦

### 2. 微量營養素（Micronutrients）

#### 維他命
| 維他命 | 功能 | 食物來源 |
|--------|------|---------|
| 維他命A | 視力、免疫 | 胡蘿蔔、菠菜 |
| 維他命C | 抗氧化、免疫 | 柑橘、番石榴 |
| 維他命D | 骨骼、免疫、情緒 | 陽光、鮭魚、蛋黃 |
| 維他命E | 抗氧化 | 堅果、種子 |
| 維他命K | 凝血、骨骼 | 綠葉蔬菜 |

#### 礦物質
| 礦物質 | 功能 | 食物來源 |
|--------|------|---------|
| 鈣 | 骨骼、肌肉功能 | 乳製品、小魚乾 |
| 鐵 | 血紅蛋白、氧氣運輸 | 紅肉、菠菜 |
| 鎂 | 肌肉放鬆、神經穩定 | 堅果、深綠蔬菜 |
| 鋅 | 免疫、傷口癒合 | 生蠔、牛肉 |
| 碘 | 甲狀腺功能 | 海帶、紫菜 |

### 3. 炎症與營養（Inflammaging）

Di Giosia et al.（2022）的研究指出：

**炎症老化（Inflammaging）**：
- 慢性低度炎症是衰老和年齡相關疾病的核心
- 營養可以顯著影響炎症水平

**抗炎食物**：
- 橄欖油（特級初榨）
- 深海魚（Omega-3）
- 藍莓、草莓（抗氧化）
- 薑黃、綠茶
- 深綠葉蔬菜

**促炎食物**：
- 加工肉類
- 精製糖
- 反式脂肪
- 過量酒精

### 4. 地中海飲食 — 最強證據支持的飲食模式

研究顯示（多次大規模RCT）：
- 降低心血管疾病風險 30%
- 降低認知衰退風險 35%
- 延長壽命 1-3 年

**核心原則**：
1. **橄欖油**為主要脂肪來源（每天 2-4 湯匙）
2. **蔬菜**為主食（每天 6+ 份）
3. **全穀物**（每天 3-5 份）
4. **魚類/海鮮**（每週 2-3 次）
5. **豆類**（每週 3+ 次）
6. **適量紅酒**（可選，每天 1 杯）
7. **限制加工肉和糖**

### 5. 飲食與衰弱/死亡率

Jayanama et al.（2021）研究發現：

- **飲食質量分數**與衰弱風險顯著負相關
- 飲食质量高 → 衰弱風險降低 40-50%
- 飲食质量高 → 全因死亡率降低 20-30%

**高質量飲食特徵**：
- 多樣性（彩虹飲食）
- 高纖維
- 低鈉、低糖、低加工食品
- 適量蛋白質

### 6. 腸道菌群與營養

- 腸道菌群被稱為「第二大腦」
- 膳食纖維餵養有益菌
- 發酵食品（乳酪、泡菜）補充益生菌
- 抗生素破壞腸道菌群平衡

## 為何重要

1. **慢性疾病預防**：心臟病、糖尿病、癌症都與飲食相關
2. **健康老化**：營養決定我們如何衰老
3. **大腦健康**：腸腦軸線 связи，飲食影響情緒和認知
4. **免疫功能**：免疫系統需要充足營養支撐

## 實際應用

- **彩虹飲食**：每天吃各種顏色的蔬果
- **211 餐盤**：2份蔬菜、1份蛋白質、1份碳水
- **間歇性斷食**：16:8 模式可能有抗衰老效果
- **廚房自制**：減少加工食品依賴
- **睡眠與飲食同步**：晚上早點吃，不吃宵夜
