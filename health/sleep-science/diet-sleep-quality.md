## 核心內容

### 1. 飲食與睡眠的雙向關係

睡眠和飲食之間存在**雙向關係**：

- 睡眠不足會影響飢餓激素（ghrelin ↑, leptin ↓），導致過度進食和對高糖高脂食物的渴望
- 飲食質量也會顯著影響睡眠結構（睡眠階段分布）和睡眠效率

這形成了一個循環：睡眠差 → 吃得更差 → 睡眠更差。

### 2. 睡眠結構基礎

正常的睡眠週期包括：

| 階段 | 特點 | 功能 |
|------|------|------|
| N1（入睡期）| 介於清醒與睡眠 | 過渡階段 |
| N2（淺睡期）| 佔總睡眠 50% | 記憶鞏固 |
| N3（深睡期）| 生長激素分泌 | 身體修復、免疫 |
| REM（快速眼動期）| 做夢發生 | 情緒處理、創意 |

飲食可以影響這些階段的時間分配和質量。

### 3. 睡眠友好飲食模式

**地中海飲食與睡眠**
- 較高的蔬果、全穀物攝入與較好睡眠質量相關
- 橄欖油中的單不飽和脂肪酸可能有保護作用
- 地中海飲食文化中的定時進食（不深夜吃）也有幫助

**高纖維飲食**
- 增加 N3 深睡眠時間
- 減少夜間覺醒次數
- 機制可能與 SCFAs 的抗發炎作用有關

**蛋白質和色胺酸**
- 色胺酸是血清素和褪黑素的前體
- 火雞、雞蛋、堅果、豆腐等富含色胺酸
- 晚餐攝入蛋白質有助於夜間恢復

### 4. 睡眠敵人：這些飲食習慣傷害睡眠

**咖啡因**
- 半衰期約 5-6 小時，下午 2 點後攝入可影響睡眠
- 不只影響入睡，也減少深睡眠（N3）和 REM 睡眠

**酒精**
- 表面上是「幫助入睡」，實際上：
  - 抑制 REM 睡眠（噩夢記憶缺失與此有關）
  - 減少 N3 深睡眠
  - 導致後半夜早醒和睡眠片段化

**高糖高脂飲食**
- 與睡眠效率降低、睡眠片段化相關
- 可能通過腸道菌群和發炎反應影響睡眠

**深夜大餐**
- 消化過程激活交感神經，不利於入睡
- 建議睡前 2-3 小時完成晚餐

### 5. 助眠食物清單

| 食物 | 有效成分 | 作用 |
|------|----------|------|
| 櫻桃/酸櫻桃 | 天然褪黑素 | 直接補充褪黑素 |
| 奇異果 | 血清素前體、抗氧化 | 改善入睡時間 |
| 核桃 | 褪黑素、omega-3 | 助眠和支持大腦功能 |
| 牛奶/溫牛奶 | 色胺酸、鈣 | 傳統助眠方法 |
| 洋甘菊茶 | 芹菜素（APigenin）| GABA 激活，鎮靜 |
| 杏仁 | 鎂、色胺酸 | 放鬆肌肉和神經 |

### 6. 實踐睡眠友好飲食原則

1. **定時進食**：每天在相似時間進食，穩定生理時鐘
2. **早餐要吃**：啟動日間節律
3. **蛋白質在早午餐**：晚間減少蛋白質以避免過度興奮
4. **限制咖啡因**：下午 2 點後避免咖啡、茶、巧克力
5. **限制酒精**：睡前 3 小時不喝酒
6. **睡前小吃**：如需，可選擇含色胺酸和碳水化合物的小食（如香蕉配坚果）

## 為何重要

1. **睡眠問題普遍**：全球約 30-45% 成年人有某種形式的失眠
2. **睡眠的健康影響**：睡眠差與肥胖、糖尿病、心血管疾病、認知退化都相關
3. **非藥物干預**：飲食調整是睡眠衛生的重要組成部分，無副作用
4. **整合醫學趨勢**：愈來愈多研究支持飲食和生活方式干預作為一線策略

## 實際應用

- 晚餐以蔬菜和全穀物為主，減少肉類和油炸食品
- 用洋甘菊或纈草茶取代晚間的酒類飲品
- 避免在電視/手機前吃零食（蓝光+食物都是睡眠敵人）
- 考慮睡前 30 分鐘喝一小杯溫牛奶
