## 核心問題

1. 電影點樣通過視覺語言講故事？
2. 剪接點樣影響觀眾嘅情感節奏？
3. 電影與其他藝術形式（文學、戲劇、音樂）嘅關係係咩？
4. 不同文化嘅電影傳統點解咁唔同？
