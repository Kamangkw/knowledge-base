## 核心內容

### 1. 為何用電影訓練同理心？

醫療專業人員需要高度同理心，但傳統醫學教育較少系統性訓練同理心能力。電影作為「安全的情感體驗場域」，讓學習者可以：

- **觀察角色情感**：從安全距離觀察患者的情感表達
- **代入他人處境**：透過角色扮演理解患者的恐懼、焦慮
- **反思自身經驗**：電影引發的情感共鳴可連結到自身醫療實踐
- **培養情感識別**：識別和命名情緒的能力

### 2. 敘事傳播理論 (Narrative Transportation)

當觀眾被電影「傳送」到故事世界時，會經歷：
- 注意力集中於情節
- 情感投入於角色命運
- 認知參與於故事推理

呢個狀態令電影比傳統講座更有效地改變觀眾的態度和信念。

### 3. 研究發現

牙科教育中使用電影進行同理心訓練的優點：

| 優點 | 說明 |
|------|------|
| 低壓力學習 | 不會對真實患者造成傷害 |
| 多元視角 | 呈現不同文化、背景患者的處境 |
| 重複觀看 | 可暫停、重播、分析細節 |
| 情感安全 | 觀眾知道是虛構，可保持情感距離 |
| 成本效益 | 比標準化病人訓練成本低 |

### 4. 推薦用於同理心訓練的電影類型

- **患者視角電影**：講述疾病經歷的電影
- **醫病關係電影**：涉及醫療決策、溝通的電影
- **生命末期電影**：涉及死亡、臨終關懷的電影
- **跨文化電影**：呈現不同文化背景患者處境的電影

---

## 為何重要

1. **醫療人文化需求**：現代醫學強調以人為本的照護
2. **同理心可訓練**：不同於智力，同理心可通過系統性訓練提升
3. **電影獨特優勢**：電影結合視覺、聲音、情感，比文字案例更強烈
4. **跨領域應用**：不只牙科，所有醫療專業都可應用
