# 音樂心理學與音樂治療

## 基本資訊

| 項目 | 內容 |
|------|------|
| 名稱 | 音樂心理學（Music Psychology）|
| 交叉學科 | 心理學、神經科學、音樂學、醫學 |
| 核心問題 | 音樂如何影響我們的情緒、認知和行為？ |
| 治療應用 | 音樂治療（Music Therapy）|

## 核心內容

### 1. 音樂與大腦

#### 音樂處理的神經網絡

聆聽音樂時，多個腦區同時激活：

| 腦區 | 功能 |
|------|------|
| 聽覺皮層 | 處理聲音 |
| 邊緣系統 | 情緒處理 |
| 伏隔核（Nucleus Accumbens）| 獎勵和愉悅 |
| 腹側紋狀體 | 預期和期待 |
| 小腦 | 節奏和運動同步 |
| 前額葉 | 審美判斷和注意力 |

**多巴胺釋放**：研究顯示，聽到「高潮」時，大腦會釋放多巴胺——這解釋了為何某些音樂讓我們「起雞皮疙瘩」。

#### 節奏同步（Entrainment）

- 大腦自然傾向於與外部節奏同步
- 這解釋了為何音樂可以影響心率、血壓、甚至行走步伐
- 節奏性聽覺刺激（RAS）可用於幫助帕金森氏症患者改善行走

### 2. 音樂與情緒

#### 情緒共鳴機制

- **情緒傳染（Emotional Contagion）**：音樂的情緒 → 聽眾的情緒
- **情節記憶**：某首歌可以讓我們瞬間回到過去的某個時刻
- **美學反應**：和諧的音樂帶來愉悅感

#### 音樂情緒的複雜性

| 音樂特徵 | 情緒反應 |
|---------|---------|
| 大調、快節奏 | 快樂、振奮 |
| 小調、慢節奏 | 悲傷、憂鬱 |
| 不和諧音 | 緊張、焦慮 |
| 低沉頻率 | 莊嚴、敬畏 |
| 高頻、上升旋律 | 希望、期待 |

### 3. 音樂對心理健康的影響

#### 音樂與壓力

- **皮質醇降低**：研究顯示，聽舒緩的音樂可以顯著降低壓力激素（皮質醇）
- **血壓降低**：放鬆音樂可以降低血壓
- **術前焦慮減少**：手術前聽音樂可以減少焦慮

#### 音樂與抑鬱

Sanfilippo et al.（2021）研究發現：

- **產前/產後心理健康**：音樂可以支持孕期和產後母親的心理健康
- 音樂干預可以減少孕期焦慮和抑鬱症狀
- 安全、無副作用，適合廣泛應用

#### 音樂與焦慮

Alneyadi et al.（2021）研究：

- **生物反饋結合音樂**：音樂 + 生物反饋技術的整合干預
- 對焦慮癥狀有顯著改善效果
- 可作為傳統治療的輔助手段

### 4. 音樂治療

#### 定義
由認證音樂治療師（MT-BC）進行的結構化治療方法，使用音樂來達成非音樂的治療目標。

#### 治療技術

| 技術 | 描述 | 應用 |
|------|------|------|
| 接受式技術 | 聆听音樂並反思 | 情緒表達、記憶刺激 |
| 主動技術 | 即興演奏、唱歌 | 溝通、自尊、社交 |
| 創作技術 | 寫詞、作曲 | 自我表達、意義建構 |
| 節奏伴奏 | 結合運動 | 運動康復、協調 |

#### 臨床應用

- **自閉癥譜系障礙（ASD）**：改善社交溝通
- **失智癥**：刺激記憶、改善情緒、減少激越行為
- **帕金森氏癥**：改善步態和平衡
- **慢性疼痛**：分散注意力、減少止痛藥需求
- **中風康復**：促進運動功能恢復

### 5. 音樂與認知功能

#### 音樂與記憶

- **情景記憶**：某首歌觸發過去經歷的記憶
- **音樂記憶**：即使在阿茲海默癥晚期，音樂記憶仍然保存
- **學習辅助**：特定音樂（巴洛克音樂）被認為可以改善專注力

#### 音樂與創造力

- 音樂訓練與創造力測試表現正相關
- 音樂可以進入「心流」狀態，促進創意思考
- 背景音樂對創造力影響混合（取決於任務類型和個人差異）

### 6. 音樂偏好與人格

研究發現音樂偏好與人格特徵的相關性：

| 音樂類型 | 人格特徵 |
|---------|---------|
| 古典、爵士 | 開放性高、創意 |
| 搖滾、重金屬 | 外向、刺激尋求 |
| 流行音樂 | 外向、親和 |
| 民謠、鄉村 | 勤勉、傳統 |
| 另類、獨立 | 創意、叛逆 |

## 為何重要

1. **非藥物治療**：音樂是安全、無副作用的輔助治療工具
2. **門檻低**：每個人都可以接觸音樂，不需要特殊技能
3. **跨文化**：音樂是人類普遍現象，適用於所有文化
4. **神經可塑性**：音樂訓練可以改變大腦結構
5. **情緒調節**：音樂是強大的情緒調節工具

## 實際應用

### 個人應用
1. **情緒管理**：建立個人「情緒播放列表」，用音樂調節情緒
2. **專注工作**：建立「專注音樂」播放列表（純器樂、無歌詞）
3. **睡前儀式**：用舒緩音樂建立睡前放鬆例程
4. **運動伙伴**：用快節奏音樂提升運動表現

### 專業應用
1. **臨床音樂治療**：由認證治療師進行
2. **安寧療護**：用音樂陪伴臨終病人
3. **教育領域**：用音樂支持特殊需求兒童
4. **企業健康**：用音樂減少工作場所壓力
