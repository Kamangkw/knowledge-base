## 日本茶道（茶道 / Sado）

### 核心精神

日本茶道是世界上最制度化的茶道形式，由禪宗僧侶村田珠光（1423-1502）開創，經武野紹鷗發展，至千利休（1522-1591）大成。

#### 四大原則

1. **和敬清寂**（Wa-Kei-Sei-Jaku）
   - **和**：和諧，人與人、人與自然的和諧
   - **敬**：敬意，對茶、對人、對當下的尊重
   - **清**：清潔，所有事物的純淨
   - **寂**：寂靜，內心的平靜和滿足

2. **一期一會**（Ichi-go ichi-e）
   - 這個茶會一生只有一次
   - 每次相會都應該以這是最後一次的心態對待
   - 提醒我們珍惜每一個當下、每一個人

3. **侘寂**（Wabi-sabi）
   - **侘**（wabi）：簡樸、不完美中的美
   - **寂**（sabi）：時間流逝、老舊、質樸
   - 反對華麗，崇尚自然、不對稱、簡素

4. **老子哲學**
   - 無為、不爭、以柔克剛
   - 茶室的極小空間體現「少則得」

### 茶室建築

日本茶室的設計處處體現哲學：

| 元素 | 哲學意涵 |
|------|---------|
| **入口（躬）** | 高度僅容一人鞠躬進入，平等你我 |
| **床（Tokonoma）** | 掛軸+花+香，唯一的裝飾焦點 |
| **地釜** | 鐵鍋煮水，模擬自然聲音 |
| **素木草席** | 不加修飾，自然建材 |

### 抹茶儀式流程

```
1. 迎客 → 客人鞠躬
2. 炭禮 → 炭組入爐，調整火候
3. 風爐 → 夏季用風爐煮水
4. 濃茶 → 主人先點濃茶（Koicha）
5. 薄茶 → 薄茶（Usucha）每人一碗
6. 懷石 → 通常茶前有簡單料理
7. 送客 → 主人送客至門口
```

## 中國茶藝

### 哲學基礎：天人合一

中國茶道根植於儒、釋、道三家哲學：

#### 道家層面
- **自然無為**：順應茶的本性
- **質樸內斂**：反對過度修飾
- **和靜清寂**：追求內心平靜

#### 儒家層面
- **以茶會友**：社交功能
- **茶禮**：待客之道
- **茶德**：廉、美、和、敬

#### 禪宗層面
- **禪茶一味**：茶即是禪
- **平常心**：日常事即修道場
- **吃茶去**：趙州禪師的「吃茶去」公案

### 功夫茶藝術

功夫茶（潮汕地區）是中國最精緻的茶道形式：

#### 器具
- 潮汕爐（小型風爐）
- 玉書碾（陶製水壺）
- 孟臣罐（宜興紫砂壺）
- 若深甌（小白瓷杯）

#### 冲泡原則
```
1. 滙壺 → 滙燙茶葉和茶具
2. 觀茶 → 賞茶葉形狀色澤
3. 高沖 → 提高水壺注水，激發香氣
4. 刮沫 → 去除表面泡沫
5. 灑茶 → 巡迴灑茶，濃度均勻
6. 聞香 → 先聞香，後品茶
```

### 中國六大茶類哲學

| 茶類 | 工藝哲學 | 性格 |
|------|---------|------|
| 綠茶 | 不發酵，自然本味 | 清爽直接 |
| 白茶 | 輕發酵，簡約 | 純粹天然 |
| 黃茶 | 悶黃工藝 | 溫和圓潤 |
| 青茶 | 半發酵，搖青 | 變化豐富 |
| 紅茶 | 全發酵 | 溫暖醇厚 |
| 黑茶 | 後發酵，陳化 | 沉穩深邃 |

## 韓國茶禮（茶道 / Darye）

### 核心精神：和敬樂

韓國茶道強調：
- **和**：自然之和
- **敬**：對茶、對人、對自然的敬意
- **樂**：品茶帶來的喜悅

### 茶禮的基本精神

源自新羅時期（公元前 57 - 935 年），融合了佛教禪宗思想：

1. **茶禪一體**
   - 喝茶是修行
   - 每一個動作都是禪

2. **五行思想**
   - 茶禮配合木火土金水五行
   - 不同季節使用不同茶和器具

3. **自然取向**
   - 户外茶會常見
   - 強調與自然的合一

### 韓國茶禮流程

```
1. 禮 → 參禮者行禮
2. 布桌 → 擺設茶具
3. 煮水 → 靜待水沸
4. 賞茶 → 觀賞茶葉
5. 溫器 → 溫熱茶具
6. 點茶 → 抹茶或散茶
7. 敬茶 → 雙手奉茶
8. 品茶 → 靜心品嚐
9. 謝茶 → 感謝主人
```

## 三國茶道比較

| 面向 | 日本 | 中國 | 韓國 |
|------|------|------|------|
| **起源** | 禪宗僧侶 | 神農氏傳說 | 新羅佛教 |
| **代表茶** | 抹茶 | 功夫茶/各類茶 | 抹茶/後发酵茶 |
| **核心精神** | 侘寂、一期一會 | 天人合一 | 茶禪一體 |
| **形式化程度** | 極高度 | 中度 | 中高度 |
| **儀式時長** | 4小時+ | 15-30分鐘 | 1-2小時 |
| **空間** | 獨立茶室 | 任何場所 | 室內外皆可 |
| **美學** | 極簡、不對稱 | 和諧、意境 | 自然、簡樸 |

## 共同教訓

### 1. 當下的力量

三國茶道都強調**此時此地**的覺知：
- 這杯茶，一生只有一次
- 吃茶去（趙州）
- 一期一會

### 2. 不完美之美

- 日本：侘寂
- 中國：殘缺美
- 韓國：自然不修

### 3. 簡單的力量

- 放下手機
- 專注泡茶
- 品嚐當下

## 為何重要？

1. **東亞文化身份**：茶道是理解中日韓文化的窗口
2. **正念練習**：無需特殊信仰，任何人都可以實踐
3. **生活美學**：將日常行為提升為藝術
4. **壓力調節**：在快節奏現代生活中提供「減速閥」

## 實踐建議

### 開始自己的茶道練習

```
1. 從一杯正念綠茶開始
   - 用你喜歡的茶
   - 完整地注水、等待、品嚐
   
2. 建立儀式感
   - 固定時間（早上或傍晚）
   - 固定地點（窗邊、陽台）
   
3. 專注於過程
   - 感受水的溫度
   - 觀察茶葉的展開
   - 注意香氣的變化
   
4. 慢慢擴展
   - 學習一種茶的沖泡
   - 嘗試功夫茶或其他形式
```
