## 摘要

本文探索一個大膽的假設：**主觀歸屬的意義（subjectively attributed meaning）構成了意識經驗的現象內容**。換言之，現象內容在本質上是語義性的（semantic）。

作者指出，這種主觀意義表現為 Qualia 的固有且非表象（non-representational）的特徵。基於現象學研究的經驗證據，作者論證：
- 主觀意義在意識經驗中無所不在
- 這個意義概念與 Frege 所說的「Sense」密切相關
- 也與 Peirce 在符號學中的「interpretant」概念一致

本文進一步在 mind-matter 關係的脈絡下，提供了與心理表徵相關的主觀意義的**形式化描述**。

---

## 主要發現（Main Findings）

### 1. 現象內容即意義（Phenomenal Content is Semantic）

本文的核心論點：

> 主觀經驗的「Qualia」不是無意義的原始感覺，而是本身就具有**意義結構**的經驗

這挑戰了傳統的「知識論鴻溝（epistemological gap）」問題——為何物理過程會產生主觀經驗？

### 2. 與 Frege Sense 的關聯

作者論證：
- Frege 的 **Sense（ Sinn）** 不僅適用於語言表達，也適用於原始感覺
- **Reference（ Bedeutung）** 對應於表象內容
- **Sense** 對應於主觀經驗的現象內容

這提供了一個橋接語言哲學與意識哲學的理論框架。

### 3. 形式化框架

作者使用數學工具（範疇論 / category theory）來形式化：

```
語法結構（Syntactic Structure）
        ↓ [解釋映射 Interpretation Map]
語義空間（Semantic Space）
        ↓ [心靈領域映射]
現象內容（Phenomenal Content / Qualia）
```

### 4. 四個層次的表象

本文呼應了 Baker 等人的框架，提出類似的四個面向：

| 面向 | 描述 |
|------|------|
| 資訊承載 | 表徵攜帶關於外部刺激的資訊 |
| 主觀有用性 | 資訊對主體有主觀意義 |
| 可用格式 | 資訊以可處理的格式編碼 |
| 意義結構 | 格式具有內在的語義結構 |

## 為何重要（Why Important）

### 1. 解決 hard problem of consciousness？

意識的「困難問題（Hard Problem）」由 David Chalmers 提出：
> 為何物理過程會產生主觀經驗？為何存在「存在起來是什麼樣子（what it is like）」的問題？

本文的貢獻在於提出：
- **困難問題可能源於錯誤的框架**
- 如果現象內容本質上是語義性的，那麼意識就不是神秘的「附加物」
- 而是資訊處理過程中自然浮現的意義結構

### 2. 連接符號學與現象學

- **Peirce 的三元關係**：符號（sign）、對象（object）、詮釋（interpretant）
- **Phenomenology**：意識的意向性結構

本文論證這兩者在本質上描述的是同一現象。

### 3. 對 AI 意識的啟示

如果意識確實基於意義結構，那麼：
- 機器意識（machine consciousness）可能不是不可能
- 關鍵在於建構具有真正語義處理的系統
- 而非僅僅模擬表面行為

## 核心概念解釋

### Qualia

**Qualia**（單數：quale）是指主觀經驗的質性特徵。例如：
- 看到紅色時的「紅色感覺」
- 品嚐蘋果時的「甜味」
- 疼痛時的「痛苦感」

這些都是無法完全用物理或功能術語捕捉的「怎樣是什麼感覺（what it is like）」。

### Frege's Sense and Reference

| 概念 | 內容 |
|------|------|
| **Reference（Bedeutung）** | 符號所指涉的客觀對象 |
| **Sense（Sinn）** | 把握參照的方式或角度 |

例如：
- 「晨星」與「暮星」有相同的 Reference（金星）
- 但有不同的 Sense（不同的把握方式）

### Non-representational Character

作者強調 Qualia 具有「非表象的」特徵：
- 傳統觀點：Qualia 表徵外部世界
- 本文觀點：Qualia **就是**意義，不是在表徵意義
- 這類似於 Sartre 對意識的描述：意識不是「關於」某物的容器，意識**就是**其意向性本身

## 與存在主義的聯繫

### Sartre 的意識理論

Sartre 在《存在與虛無》中提出：
- **意識沒有「內在」**：意識總是「意識到某物」
- **自發性（spontaneity）**：意識是主動建構的，而非被動接收
- **虛無（néant）**：意識能夠否定、質疑、超越當下情境

本文的「意義即現象內容」與 Sartre 的意識理論驚人地呼應：
- 意識不是空洞的容器，而是充滿意義的主動活動
- Qualia 不是被動的感受，而是有意義的經驗

### Camus 與荒謬

Camus 的「荒謬」概念：
- 人类對意義的追求與冷漠宇宙之間的矛盾
- 生活的意義不是給定的，而是需要創造的

本文的框架提供了一個可能的解答：
- 意義不是外在的，而是內在於意識結構中
- 即使在「荒謬」的情境中，我們的經驗本身就帶有意義

## 批評與限制

1. **形式化是否充分？**：數學形式化是否真的捕捉了主觀經驗？
2. **經驗證據不足**：現象學研究的證據是否足够嚴格？
3. **意識的困難問題**：本文是否真正解決了 hard problem？
4. **爭議性的「非表象」概念**：許多哲學家會質疑 Qualia 是否真的非表象
