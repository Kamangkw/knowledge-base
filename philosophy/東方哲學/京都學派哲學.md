# 京都學派哲學

## 基本資訊

- **名稱**：京都學派（Kyōto-gakuha / Kyoto School）
- **創始年份**：1913年，西田幾多郎於京都帝國大學
- **地理中心**：日本京都大學
- **時間跨度**：1910年代至今，橫跨四個世代
- **核心概念**：絕對無（zettai-mu / Absolute Nothingness）
- **主要特色**：將東亞思想（特別是佛教、道家）與西方哲學進行批判性及創造性對話
- **主要代表人物**：西田幾多郎、田邊元、西谷啟治、上田閒照等

---

## 1. 歷史背景與命名由來

### 1.1 一個非典型的哲學流派

京都學派與西方哲學史上的大多數流派不同，它從未成立正式的學院、學會或機構。與法蘭克福學派有馬庫色、本雅明等明確成員不同，與布魯姆學派有明確的師承譜系也不同，京都學派更像是一個「以京都大學為中心的非正式思想網絡」。

這個「學派」的成員在二戰前、二戰期間和戰後有截然不同的政治與哲學立場，有些成員甚至相互批評。這使得京都學派的身份認同問題至今仍是學術界熱烈討論的話題。

### 1.2 名稱的偶然性

「京都學派」這個名稱是1932年由**戶田坂純**（Tosaka Jun）首次提出，當時是一個帶有批評意味的標籤，指責西田幾多郎和比他年輕的田邊元是「資產階級哲學」的代表。戰時記者使用這個名稱描述在京都舉行的哲學研討會，因而與軍國主義時期的政治爭議產生了複雜的關聯。

直到1970年代，「京都學派」才成為這個哲學運動的標準名稱，被國際學術界廣泛接受。此時距離西田發表《善的研究》已經過去了六十多年。

### 1.3 產生的歷史語境

明治維新（1868年）後，日本在現代化過程中全面引進西方哲學東京帝國大學的哲學系成為德國觀念論的傳播中心，許多日本哲學家留學德國，回國後傳播康德、黑格爾的思想。

在此背景下，西田幾多郎的創新在於：他不是簡單地接受或轉述德國觀念論，而是將其與禪宗佛教、道家思想和日本傳統美學進行深層融合，創造出真正具有東亞特色的哲學體系。

## 2. 核心哲學家

### 2.1 西田幾多郎（Kitarō Nishida, 1870–1945）：創始者

西田幾多郎被譽為「近代日本哲學之父」，他的哲學生涯經歷了三個階段的演變：

**第一期（早期）：純粹經驗**
- 受威廉·詹姆斯（William James）「純粹經驗」概念的啟發
- 主張哲學的起點是超越主客二分之前的原初經驗
- 1911年發表《善的研究》，奠定其哲學基礎

**第二期（中期）：場所的邏輯**
- 發展出「場所」（basho / 場所）的概念
- 提出「絕對無的場所」——這是使一切存在得以顯現的先驗根據
- 引入「絕對矛盾的自我同一」概念，挑戰黑格爾的正反合辯證法

**第三期（晚期）：最後著作**
- 深入宗教與形上學的關係
- 1945年發表《宗教與宗教世界觀》

西田的哲學終極關懷是：如何克服西方哲學中根深蒂固的主客對立？如何從東亞的「無」的智慧中找到一種新的哲學方向？

**西田的「場所邏輯」核心要點：**

> 現實是存在，同時也是無；存在而又無，無而又存在；現實是主觀與客觀的統一，是絕對矛盾之物的自我同一。

在「場所」中，對立的兩極（A與非A）並非被消滅（像黑格爾那樣），而是保持在張力之中。這種張力是動態的、永恆的——矛盾永遠是矛盾，但被包容在一個更大的整體之中。

### 2.2 田邊元（Hajime Tanabe, 1885–1962）：批評性繼承者

田邊元是西田的學生，但他的哲學方向與西田有顯著分歧。田邊批評西田的哲學太過抽象，缺乏對具體歷史與社會的關注。

**田邊的核心概念：**

- **種的邏輯（Logik der Art）**：田邊認為西田的「場所」太過靜態，他提出以「種」（species）概念取而代之
- **悔恨的邏輯**：戰後田邊公開懺悔，認為他的「種的邏輯」被軍國主義者利用，轉向以「悔恨」為核心的哲學
- **宗教哲學的轉向**：晚期田邊更強調悔恨與救贖的主題

田邊的「種的邏輯」在戰時被軍國主義意識形態利用，是京都學派最黑暗的歷史教訓。田邊本人在戰後多次公開懺悔，指出「種的邏輯」被政治濫用的原因。

### 2.3 西谷啟治（Nishitani Keiji, 1900–1990）：戰後高峰

西谷啟治是戰後京都學派最重要的代表，1982年發表的《宗教與虛無主義》是京都學派的巔峰之作。

**西谷的核心貢獻：**

- 將「絕對無」與尼采的「上帝已死」、海德格的存在問題進行深度對話
- 區分「空」與「虛無」：空不是什麼都沒有，而是「能夠包容一切的充實」
- 探討在徹底的無神論處境下，宗教意義如何可能

### 2.4 上田閒照（Ueda Shizuteru, 1926–2019）：現象學與禪的交匯

上田閒照是戰後最重要的小提琴，將禪宗思想與現象學進行系統性比較：

- 探索「二」（主客對立）與「無」的關係：禪的開悟不是認識論的飛躍，而是從「二」的立場回歸到「無」
- 對海德格的「此在」（Dasein）與禪的「無」進行創造性比較

## 3. 核心哲學概念：絕對無（Zettai-mu）

### 3.1 三種「無」的層次

京都學派哲學的核心是對「無」的精密分析。理解「絕對無」需要先區分三個層次：

| 層次 | 日文 | 內涵 |
|------|------|------|
| **相對無** | 相對的無 | 對某物的否定，如「盒子裡沒有蘋果」 |
| **虛無/虛無主義** | 虛無 | 對存在本身的否定，如「一切都是虛無」 |
| **絕對無** | 絶対無（zettai-mu） | 不是對存在的否定，而是一切存在得以成立的根據 |

「絕對無」的「絕對」意味著：它不是相對於「有」而存在的，而是「有」與「無」兩者得以成立的先驗場所。

### 3.2 佛教淵源：大乘空宗

- ** Śūnyatā（空）**：大乘佛教的核心概念，日文為「空」（kū）
- **緣起**（pratītya-samutpāda / engi）：一切法因緣而生，無自性（svabhāva）
- **龍樹（Nāgārjuna）**的深刻命題：「涅槃的界限，就是輪迴的界限」——解脫不在於逃離輪迴，而在於在輪迴之中看見空性
- 真正的空不是否定一切，而是「能動地否定存在與相對無之對立本身」

### 3.3 道家淵源：《道德經》與《莊子》

- **《道德經》第40章**：「天下萬物生於有，有生於無」
- **《道德經》第14章**：「視之不見名曰夷，聽之不聞名曰希……是謂無狀之狀，無物之象，是謂惚恍」
- **《莊子》**的「坐忘」——忘記知識、仁義、聖智，最後忘記自己的存在，心如明鏡

### 3.4 與西方「無」的傳統比較

| 傳統 | 對「無」的理解 |
|------|-------------|
| **海德格** | 「無」是我們對存在者整體進行發問時顯現的 |
| **黑格爾** | 無是絕對理念運動的起點，最終被揚棄在絕對精神中 |
| **田邊元** | 絕對無是一種宗教性體驗，與悔恨和救贖相關 |
| **西田幾多郎** | 絕對無是場所，一切對立在其中保持張力 |

## 4. 與歐洲存在主義的交匯

京都學派與歐洲存在主義有深刻的哲學親緣性，但也有根本差異：

### 4.1 與海德格的對話

- 雙方都對西方形上學傳統進行根本性批判
- 海德格的「存在的遺忘」vs 西田的「無的發現」
- 海德格晚期「存在之讓渡」（Gelassenheit / 泰然任之）與禪宗的「放下」驚人相似

### 4.2 與尼采的對話

- 西谷啟治直接處理尼采「上帝已死」的命題
- 「上帝已死」意味著最高的價值不再存在，人必須自己創造意義
- 但京都學派認為，尼采的「強力意志」仍然是「有」的範疇，真正的超越是進入「無」

### 4.3 與沙特的比較

- 沙特的「存在先於本質」與西田的「場所」哲學有結構相似性
- 但沙特強調人的絕對自由與選擇責任
- 西田強調在「無」中的放下與包容，而非主動創造

## 5. 批評與歷史局限

### 5.1 戰時政治的污點

這是京都學派最嚴重的歷史問題：

- 田邊元的「種的邏輯」被軍國主義利用，為日本帝國主義提供了「哲學」論證
- 多位核心成員對戰時政策保持沉默或表示支持
- 戰後的悔恨與清算成為京都學派成員集體面對的問題

### 5.2 概念的含糊性

- 「絕對無」究竟是認識論概念、本體論概念，還是先驗哲學概念？學界並無共識
- 批評者認為「絕對無」不過是禪宗公案的哲學翻譯，缺乏真正的概念精確性

### 5.3 與宗教的關係問題

京都學派的哲學是否本質上是宗教哲學？這個問題困擾著對它的學術評價。部分哲學家認為，真正的哲學應該保持世俗性，而不應該像京都學派那樣與佛教禪宗有如此緊密的關聯。

## 6. 當代意義（2026年視角）

2026年夏季學校由大橋良介主持，主題延續京都學派對東西方哲學對話的探索。在AI與全球化的語境下，京都學派的當代價值：

- **對AI意識問題的啟示**：京都學派對「無」的思考，有助於區分人類意識與機器計算的根本差異
- **東西方真正的哲學對話**：京都學派不是用西方框架詮釋東方，而是創造性地融合兩者
- **現代性的另類可能**：在歐洲啟蒙現代性之外，展示了亞洲現代性的哲學路徑
- **對當代虚无主义的回应**：在價值虛無主義盛行的時代，京都學派的「空」提供了一種不陷入虛無的出路
