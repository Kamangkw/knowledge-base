# 存在主義哲學

## 基本資訊

| 項目 | 內容 |
|------|------|
| 名稱 | 存在主義（Existentialism） |
| 起源 | 19世紀丹麥哲學家齊克果（Søren Kierkegaard），20世紀由海德格、薩特發揚 |
| 主要代表 | 齊克果、尼采、海德格、薩特、卡繆、馬塞爾 |
| 核心議題 | 生命的意義、自由、責任、孤獨、死亡 |

## 核心內容

### 1. 「存在先於本質」（Existence Precedes Essence）

這是存在主義最核心的命題。薩特（Jean-Paul Sartre）提出：

> 「存在先於本質」—— 人首先是存在著，然後才定義自己是什麼。

這意味著：
- 人沒有固定的「本質」或「天性」
- 人的身份是自己透過選擇和行動創造出來的
- 沒有預設的價值觀或目標等待我們去實現

### 2. 自由的重量（The Weight of Freedom）

存在主義者認為人類是絕對自由的，但這種自由同時是一種沉重的責任：

- 我們無法逃避選擇，即使「不做選擇」本身也是一種選擇
- 自由帶來焦慮（angst）和不安
- 個人必須為自己的選擇承擔全部責任

海德格（Martin Heidegger）稱這種狀態為「被拋入世界」（Thrownness / Geworfenheit）—— 我們沒有選擇是否來到這個世界，但我們必須對如何活著負起責任。

### 3. 存在的焦慮（Existential Angst）

面對自由和虛無，人會經歷一種獨特的存在焦慮：

- **面對死亡**：海德格的「向死而生」（Being-towards-death）
- **面對虛無**：尼采的「上帝已死」象徵傳統價值的崩潰
- **面對孤獨**：在宇宙中沒有外在意義的支撐

卡繆（Albert Camus）在《荒謬之旅》中描述這種荒謬感：人類對意義的渴求與宇宙的沉默之間的對立。

### 4. 虛無主義的對峙

尼采是第一位正面面對虛無主義的哲學家：

> 「上帝已死」—— 標誌著傳統宗教和道德體系的崩潰

存在主義與虛無主義不同：
- 虛無主義：既然沒有客觀意義，就什麼都不重要
- 存在主義：沒有預設意義，但人可以自己創造意義

### 5. authentic vs. inauthentic existence

海德格區分兩種生存模式：
- **真實存在（Authentic）**：直面死亡，承擔自由，活出真正的自我
- **非真實存在（Inauthentic）**：逃避自由，融入「常人」（Das Man），逃避責任

## 為何重要

1. **現代人的精神困境**：在傳統宗教權威消退後，如何找到生命的意義？
2. **心理治療的基礎**：維克多·弗蘭克爾（Viktor Frankl）的意義治療（Logotherapy）直接源於存在主義
3. **文學與藝術**：卡夫卡、卡繆、村上春樹等作家深受存在主義影響
4. **對抗現代消費主義**：提醒我們不被社會期待定義，而是主動選擇

## 實際應用

- **心理諮商**：協助個案面對生命意義危機
- **生涯規劃**：在沒有固定答案的世界中，主動定義自己的價值
- **創意寫作**：探索人物的內心掙扎和成長
- **領導力發展**：接受不確定性，在模糊中做出決策
- **面對衰老與死亡**：以「向死而生」的態度活出有意義的人生
